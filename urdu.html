6. SEO kya hota hai (Search Engine Optimization)

Jab bhi hum koi website banate hain to hum chahte hain ke wo Google ya kisi bhi search engine par dikhai de. Lekin ye tabhi mumkin hota hai jab hum apni website ka SEO theek se karein. SEO ka matlab hota hai Search Engine Optimization. Ye aise tareeqe hain jinke zariye hum apni website ko search engine ke liye behtar banate hain taa ke wo upar dikhai de.

SEO kyu zaroori hai?

Agar aap chahte ho ke log aapki website tak asani se pahunch sakein to SEO karna bohot zaroori hai. Agar SEO nahi karte to aapki website search result mein peeche chali jati hai aur log use dekh nahi pate.

SEO ke basic points

  • Website ke title ko hamesha unique aur clear rakho.
  • meta description likho taa ke Google samjhe ke page kis baare mein hai.
  • Keywords use karo jo log search karte hain.
  • Website fast load honi chahiye.
  • Images mein alt attribute likhna zaroori hai.

Example


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Awesome Website</title>
    <meta name="description" content="This website teaches HTML in Urdu and English in a very simple way.">
    <meta name="keywords" content="HTML, Urdu HTML, Learn HTML, Web Design">
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is an example of SEO in HTML.</p>
</body>
</html>

Explanation

Is main meta tag ke andar attributes use kya hai name="" is main ham ne ek jagha description dala hai ye chrome ko batata hai ke hamari website kis bare main hai content ke andar phir description likhna hai.name="Keywords" ye ye karta hai ke agar koy search kare Learn HTML in Urdu to yaha Keywords use hoyr hai jese Learn Urdu to Keywords ye karta hai ke batata hai chrome ko ke agar koy ye Keywords search kare to hamari website dikha dena.
